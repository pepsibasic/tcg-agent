Implement `summarizePortfolio(userId: string)` in `packages/agent/src/orchestrators/portfolio-summary.ts`.

Define return types:
```typescript
type PortfolioSummarySuccess = { success: true; data: PortfolioSummaryResponse; degraded?: boolean }
type PortfolioSummaryResult = PortfolioSummarySuccess
```

Pipeline:
1. **Fetch all user cards:** `prisma.userCard.findMany({ where: { userId, deletedAt: null }, include: { card: true } })` and `prisma.externalCard.findMany({ where: { userId, deletedAt: null } })`.

2. **Compute DB-derived fields deterministically:**
   - `totalValueEst`: Sum of `Number(uc.estimatedValue ?? 0)` for all userCards + `Number(ec.estimatedValue ?? 0)` for all externalCards.
   - `breakdown`: Group all cards by `ipCategory` (for userCards: `uc.card.ipCategory`, for externalCards: use `'External'` or try to infer from name — simplest approach: treat all externalCards as ipCategory `'External'`). For each group: `{ ipCategory, totalValue, cardCount, percentOfPortfolio }`.
   - `priceDataAsOf`: Most recent `priceFetchedAt` across all cards, as ISO string. Null if none.
   - `priceConfidence`: Worst confidence across all cards (NO_DATA > STALE_7D > RECENT_24H > LIVE). If any card is NO_DATA, overall is NO_DATA.

3. **Render prompt:** `renderPrompt('portfolio_summary', { user_id: userId, cards_json: JSON.stringify(breakdown), total_count: totalCards.toString(), vaulted_count: userCards.filter(c => c.state === 'VAULTED').length.toString(), external_count: externalCards.length.toString() })`.

4. **Call LLM:** `generateWithRetry({ schema: PortfolioSummarySchema, prompt, system, config: DEFAULT_LLM_CONFIG, narrativeFields: ['recommendedActions', 'missingSetGoals'] })`. Note: the LLM produces the full PortfolioSummarySchema shape (including concentrationScore, liquidityScore, collectorArchetype). The orchestrator then overrides the DB-computed fields with its own calculations.

5. **Success path:** Merge LLM output with DB-computed fields: `{ ...llmResult.data, userId, totalValueEst, breakdown: dbBreakdown, priceDataAsOf, priceConfidence: overallPriceConfidence }`. The LLM's concentrationScore and liquidityScore are trusted (it sees the full cards_json).

6. **Degraded path:** Return deterministic defaults: `{ userId, totalValueEst, breakdown: dbBreakdown, concentrationScore: 0, liquidityScore: 0, collectorArchetype: null, missingSetGoals: [], recommendedActions: [], priceDataAsOf, priceConfidence: overallPriceConfidence, degraded: true }`.

**Update barrel export:** Add `export { summarizePortfolio } from './portfolio-summary.js'` to `packages/agent/src/orchestrators/index.ts`.

- Always convert Prisma Decimal to Number before arithmetic: `Number(card.estimatedValue ?? 0)`
- Use `'recommendedActions'` (not `'recommended_actions'`) as narrativeField name — must match the actual field name in the Zod schema object
