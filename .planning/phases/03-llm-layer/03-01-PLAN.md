  1. Create `packages/agent/src/llm/types.ts`:
     - Export `LLMProvider` type: `'openai' | 'anthropic'`
     - Export `LLMProviderConfig` interface: `{ provider: LLMProvider, model: string, apiKey?: string, maxRetries?: number, temperature?: number }`
     - Export `GenerateOptions<T>` interface: `{ schema: z.ZodSchema<T>, prompt: string, system?: string, config: LLMProviderConfig }`
     - Export `AnalysisFailure` type: `{ status: 'failed', reason: string, partial: Record<string, unknown> | null, retryable: boolean }` — designed to be reusable by all three Phase 4 orchestrators
     - Export `GenerateResult<T>` = `{ success: true, data: T } | { success: false, failure: AnalysisFailure }`

  2. Create `packages/agent/src/llm/sanitize.ts`:
     - `sanitizeInput(value: string | null | undefined): string` — XML-escapes `<>&'"` characters, returns '' for nullish
     - `wrapUserInput(type: string, value: string): string` — calls sanitizeInput then wraps: `<user_input type="${type}">${escaped}</user_input>`
     - Export both functions

  3. Create `packages/agent/src/llm/client.ts`:
     - `import { generateObject } from 'ai'`
     - `import { openai } from '@ai-sdk/openai'`
     - `import { anthropic } from '@ai-sdk/anthropic'`
     - `function getModel(config: LLMProviderConfig)` — returns `openai(config.model)` or `anthropic(config.model)` based on provider field
     - `async function generateStructured<T>(options: GenerateOptions<T>): Promise<GenerateResult<T>>` — calls `generateObject({ model: getModel(options.config), schema: options.schema, prompt: options.prompt, system: options.system })`, returns `{ success: true, data: result.object }` on success. On ZodError or AI SDK error, returns `{ success: false, failure: { status: 'failed', reason: error.message, partial: null, retryable: true } }`
     - Export `generateStructured` and `getModel`

  4. Install Vercel AI SDK dependencies:
     - `pnpm add ai @ai-sdk/openai @ai-sdk/anthropic --filter @tcg-agent/agent` (or correct package name from package.json)
